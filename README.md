# simple-python-blockchain

Cada bloco neste script é gerado a partir da combinação dos seus próprios dados com a assinatura digital (hash) do bloco que o antecede. Se qualquer informação no Bloco 1 for alterada, o seu hash muda, quebrando o link com o Bloco 2 e invalidando toda a sequência.

## Como rodar
Apenas certifique-se de ter o Python 3 instalado e execute:
python3 blockchain.py
